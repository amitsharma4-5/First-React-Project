# First React Project 🚀

A modern frontend project built with **Vite**, **JSXPine**, **Alpine.js**, and **Tailwind CSS**.

## Tech Stack

- [Vite](https://vite.dev) — Fast build tool
- [JSXPine](https://www.jsxpine.com/core) — JSX + Alpine.js component library
- [Alpine.js](https://alpinejs.dev) — Lightweight reactive JS framework
- [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS framework

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── ui/
│   ├── components/     # Reusable UI components (Button, Badge, Card...)
│   └── scripts/        # Alpine.js directives and magic helpers
├── views/              # Page-level views (Counter, Greeting...)
├── utils/              # Helper utilities
└── styles/             # Global CSS
```

## Features

- ⚡ Counter with Alpine.js reactivity
- 👋 Greeting input view
- 🌙 Dark / Light theme toggle
- 🧩 Reusable Badge, Card, Footer components
- 🛠️ Utility helpers (capitalize, clamp, formatCount)
